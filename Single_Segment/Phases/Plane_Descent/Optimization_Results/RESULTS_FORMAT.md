# VTOL Optimization Results Format

## MAT File Contents
- **V**: Airspeed [m/s]
- **gamma**: Flight path angle [rad]
- **alpha**: Angle of attack [rad]
- **epsilon**: Thrust vector angle [rad]
- **n**: Motor RPM
- **T**: Total thrust [N]
- **phi**: Effective thrust angle [rad]
- **L**: Lift force [N]
- **D**: Drag force [N]
- **CP**: Power coefficient
- **J**: Advance ratio
- **xf**: Horizontal distance [m]
- **L_over_W**: Lift-to-weight ratio
- **T_over_W**: Thrust-to-weight ratio
- **L_over_D**: Lift-to-drag ratio
- **timestamp**: Simulation timestamp
- **software_version**: Code version
- **param_config**: Full parameter structure
